# Code of Conduct

## Our Commitment

We are committed to maintaining a respectful, constructive, and inclusive environment for everyone contributing to Cultivate.

## Expected Behavior

Contributors are expected to:
- Be respectful in communication
- Assume good intent while giving clear feedback
- Focus critique on the work, not the person
- Use welcoming and professional language
- Respect different levels of experience

## Unacceptable Behavior

The following behavior is not acceptable:
- Harassment, insults, or personal attacks
- Discriminatory or exclusionary language
- Repeated disruptive or hostile conduct
- Publishing private information without consent
- Bad-faith issue filing or spam submissions

## Scope

This Code of Conduct applies to:
- GitHub issues
- Pull requests
- Discussions related to this repository
- Any external contribution process connected to this project

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that do not align with this Code of Conduct.

Repeated or severe violations may result in temporary or permanent restrictions on participation.

## Reporting

If you encounter behavior that violates this Code of Conduct, please report it to the maintainers through the team’s designated communication channel and include relevant context or screenshots when possible.
